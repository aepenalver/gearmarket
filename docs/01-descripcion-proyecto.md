# 01 · Descripción del proyecto

## Nombre
**GearMarket**

## Tipo de proyecto
Marketplace web especializado en equipamiento deportivo.

## Problema que resuelve
Las plataformas genéricas para compra y venta de productos usados no están especializadas en deporte.  
Eso genera:
- publicaciones desordenadas
- poca claridad para comparar productos
- filtros poco útiles para artículos deportivos
- menor confianza al buscar equipamiento específico

## Público objetivo
Personas entre 18 y 45 años que practican deporte y desean comprar o vender equipamiento deportivo usado o seminuevo:
- ciclismo
- running
- fitness
- outdoor
- entrenamiento funcional

## Propuesta de valor
Permitir que deportistas compren y vendan equipamiento deportivo de manera simple, rápida y organizada, dentro de un marketplace especializado.

## Alcance inicial
- registro e inicio de sesión
- galería de publicaciones
- detalle de producto
- perfil de usuario
- creación de publicaciones
- favoritos
- contacto con vendedor
